Assignment A2 
=============================

Install compiler and libraries used by the code:

```bash
sudo apt-get update && sudo apt-get install -y libssl-dev
```

Q1 
-----------------------
Compile:

```bash
g++ -O3 q1.cpp -o q1 -lcrypto 
```

Run:

```bash
./q1
```

To increase/decrease runtime, edit the `SEARCH_LIMIT`, `COL_SEARCH_LIMIT` and related constants near the top of `q1.cpp`.


Q2 / Q3
---------------------------------------------------
Compile:

```bash
g++ -O3 q2.cpp -o q2 -lgmpxx -lgmp
g++ -O3 q3.cpp -o q3 -lgmpxx -lgmp
```

Run:

```bash
./q2
./q3
```

Outputs
-------------------------
All outputs are printed to the console. One run of output for each question is included in the `outputs` directory. The solution to Q1 and Q2 is in `q1.cpp` and `q2.cpp` respectively. The solution to Q3 part (a) is in `part3_a.pdf` and the solution to remaining parts is in `q3.cpp`.
